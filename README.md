# Vanta
A KoGaMa Theme inspired by a gradient color scheme found in [this video](https://www.youtube.com/watch?v=v-nv_ZVDSb0). <br>
A summer sunset done right. 

### Features
AllowPaste <br>
ProcessIMGRGB <br>
DeBloatNavBar <br>

And then pure CSS with lots of subtle but dynamic feel.
